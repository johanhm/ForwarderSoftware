/*! \mainpage XT28 Forwarder API Introduction
 
 Welcome to the API documentation for the XT28 Forwarder API.
 This API uses the bodas API to implement its functionality.
 The API contains of the following main moduels:
 
 - \ref ADC
 - \ref EXP
 - \ref PAActuate
 - \ref Sensors
   + \ref IMU
   + \ref PAPOS
   + \ref PAPR

 
 Related hardware constants for the XT28 is defined in the headerfile of
 - \ref XT28HWC
 
 Sensor modules and Actuate moduels have CAN support for sending out sensor values on CAN. They requiere the use of the following addon
 - \ref CANSuport
