Get the reference signal for a wheel given a force reference, current measured force, pressure on both chambers and the velocity for a wheel. This function have the option of dead band which means If true the output signal will turn zero if the error changes sign or becomes zero. The signal will activate again when the error is greater then 10%.

The return value is the suggested flow on the cylinder. Positive means the wheel should go down, negative means it should go up.

Parameters
[in]wheelWheel number
[in]messuredCylinderForceMeasured force
[in]pesudoForceReference_NReference force
[in]pressureA_PaPressure on chamber A for the wheel
[in]pressureB_PaPressure on chamber B for the wheel
[in]velocityWheel_msVelocity for the wheel
[in]deadbandStateSet if the signal should dead band or not.
Returns
Flow reference in % [-1, +1]
