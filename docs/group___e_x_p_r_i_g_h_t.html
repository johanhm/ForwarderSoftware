Excipad right
Driver interface

Description of module

Right excipad buttons and joystick.

Does the following:

  1. Setup right excipad
  2. Get what button driver have pressed
  3. Get joystick value.

void EXPConfigureExcipad ( uint8  CANChannel,
int  buttonDataboxNr,
int  joystrickDataboxYNr,
int  joystickDataboxXNr 
)

This function is mandatory to call in the setup of the program if you need to use this module. This function sets up what channel the Excipad is connected. The setup require to use two data boxes in the BODAS C-API. Any channels can be used that is valid by the BODAS API and is not used by any other channel.

Parameters
[in]CANChannelThe CAN channel the Excipad is connected.
[in]buttonDataboxNrA data box number, valid range is [1 - 100]
[in]joystrickDataboxNrA data box number, valid range is [1 - 100]
