Cabin Sensors
Sensors

Description of module

Handles sensors related to the cabin.

Does the following:

  1. Configure cabin sensors
  2. Update sensor data
  3. Get the chair position.

The module handles the following sensors

  1. Chair position
  2. Oil temperature
  3. Door sensor
  4. Fuel meter sensor
