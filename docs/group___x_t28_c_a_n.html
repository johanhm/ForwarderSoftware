CAN
Support

Description of module

CAN related functionality. This module does the following:

  1. Set up CAN channels with XT28 settings
  2. Provide user with functions for sending signed and unsigned int16
  3. Send Battery voltage and Debug variables on CAN

Global debug variables

For any module that import CAN Support its possible to use these global variables to send any data on CAN. To send the data of global debug variables the user have to call CANSendDebuggMessage(uint8 CANChannel) periodically.
